# sixins
用于学习SpringBoot,Netty,fastDFS的聊天app

服务器环境搭建参考个人博客:https://www.jianshu.com/p/282f6c02d878

技术选用:
后端 SpringBoot 2.0
消息通信 Netty 4.x
数据库:Mysql/MariaDB
文件存储: Nginx+基于fastDFS搭建的图片分布式服务器
持久层:Mybatis
服务器:腾讯云
